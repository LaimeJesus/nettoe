-- $Id: INSTALL file for netToe version 1.5.1 --
------------------------------------------------

### In terminal run

    tar -zxvf nettoe-1.5.1.tar.gz
    cd nettoe-1.5.1

    ./configure
    autoreconf -i
    make
    sudo make install
